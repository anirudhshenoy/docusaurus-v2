# app.allFunctions

Created by Harish Mukkara

## Use Case

This will list all the functions built for bot

```javascript
let allFunctions = app.log(Object.keys(app.allFunctions));

// the log will be
[
 main,
 didYouMean,
 .
 .
 allcloudfuctions
 .
]
```
